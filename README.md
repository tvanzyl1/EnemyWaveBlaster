# Neon Necro Blaster

`Neon Necro Blaster` is a single-file browser survival shooter built in plain HTML, CSS, and JavaScript.

## What It Includes

- Top-down zombie wave survival in one `index.html`
- Auto-aim and auto-fire combat inspired by horde survival arcade games
- Multiple zombie types, XP gems, health pickups, bonus crates, and upgrade choices
- Mobile virtual stick support plus keyboard controls
- Juice-heavy feedback with particles, floating text, flashes, explosions, and screen shake

## How To Run

Open `index.html` in a browser.

If your browser blocks local file execution features, serve the folder with a basic local server instead.

## Controls

- `WASD` or arrow keys: move
- Mobile: drag the on-screen virtual stick
- Aiming and firing are automatic

## Notes

- Best score is stored with `localStorage`
- The project intentionally uses no external libraries
- The code is grouped into clear sections inside the single HTML file: state, entities, update loop, rendering, and UI
